## How to clone and run project

### cloning repo
```console
git clone https://github.com/Diegodossantos23/angular-crud.git 
```

###change directory to "angular-crud"
```console
cd angular-crud
```

### how to run backend
```console
cd backend
npm start
```

### how to run frontend

```console
cd frontend
npm start
```
