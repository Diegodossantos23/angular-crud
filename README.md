<h3> How to clone and run project</h3>

<p>cloning repo</p>
```console
git clone https://github.com/Diegodossantos23/angular-crud.git 
```

<p>change directory to "angular-crud"<p/>
```console
cd angular-crud
```

<p>how to run backend</p>
```console
cd backend
npm start
```

<p>how to run frontend</p>
```console
cd frontend
npm start
```
