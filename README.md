## How to clone and run project

### Hoe to clone repo:
```console
git clone https://github.com/Diegodossantos23/angular-crud.git 
```

### Change directory to "angular-crud":
```console
cd angular-crud
```

### How to run backend:
```console
cd backend
npm start
```

### How to run frontend:

```console
cd frontend
npm start
```
